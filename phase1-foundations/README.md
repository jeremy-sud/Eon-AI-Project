# Fase 1: Fundamentos (Python)

Esta fase contiene el prototipo lógico y conceptual del motor Eón. Escrito en Python para facilitar la experimentación rápida y la visualización.

## Contenido

- `esn/`: Implementación de Echo State Network en Python puro.
- `notebooks/`: Jupyter Notebooks con experimentos de caos y predicción.

## Ejecución

1. Crear entorno virtual:

   ```bash
   python -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
   ```

2. Correr demo:
   ```bash
   python esn/esn.py
   ```
