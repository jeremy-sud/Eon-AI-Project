LÍNEA DE TIEMPO DEL PROYECTO EÓN

Desarrollo progresivo y sistemático del sistema Eón AI, desde sus cimientos matemáticos en Python hasta la arquitectura de modelos autorregresivos colaborativos.

Fase 1: Foundations (Fundamentos) Completado
Implementación matemática del Echo State Network (ESN) en Python. Definición de matrices recurrentes, inicializadores, radio espectral y regresión Ridge.
Fase 2: C Core (libAeon C Library) Completado
Reescritura de las rutinas de simulación matricial y activación neuronal en C puro para un rendimiento nativo. Construcción de extensiones de Python y bindings.
Fase 3: Integration (Arquitectura Completa) Completado
Acoplamiento del servidor web Flask, el broker MQTT (Mosquitto) y el puente WebSocket (ws-bridge) para telemetría bidireccional en tiempo real con el navegador.
Fase 4: Hardware Integration (Edge Computing) Completado
Algoritmos de procesamiento analógico a digital para señales físicas en microcontroladores periféricos, conectándose de forma distribuida a la red local.
Fase 5: Applications (Oráculo & Alquimia) Completado
Diseño de la consola de transmutación de elementos (Nigredo, Albedo, Rubedo) y del oráculo neuronal I-Ching que predice hexagramas usando el estado interno del reservorio.
Fase 6: Collective Mind (Mente Colectiva) Completado
Creación del protocolo de compresión de pesos 1-Bit (comprimiendo 17x) y el procesador homeostático de la mente grupal (Egrégor) para coordinación federada sin datos centralizados.
Fase 7: Language (Modelos de Lenguaje TinyLM) Completado
Diseño de TinyLMv2, un modelo autoregresivo basado en ESN para generación de texto, y el orquestador colaborativo multi-nodo para el chat interactivo de Eón.

GUÍA DE LA INTERFAZ DE EÓN AI
¿Cómo funciona el Chat Colaborativo de Eón?
Cuando escribes un mensaje, este se envía en paralelo a tres nodos especializados del reservorio (Intención, Respuesta y Coherencia). Cada nodo procesa el texto y modula el estado dinámico general. Luego, el motor TinyLMv2 genera los tokens de la respuesta final ajustándose a la personalidad y la retroalimentación del Egrégor colectivo.
¿Qué representa el gráfico de "Eón Dream"?
Eón Dream es una ventana directa a la "mente" del sistema. En el modo **Grid** ves el estado de activación analógico de las 100 neuronas del reservorio en tiempo real. En el modo **Path** puedes seguir la trayectoria caótica de las dos dimensiones principales en el espacio de fase. Si desactivas el **Pulse**, las neuronas responderán a ruido aleatorio de fondo en lugar de a ondas estructuradas.
¿Qué es la "Transmutación Alquímica"?
Es una analogía interactiva sobre las fases de optimización y consolidación de un sistema neuronal:
• Nigredo (Disolución): Introduce caos y perturba los pesos para explorar el espacio de pérdida.
• Albedo (Purificación): Normaliza y regulariza los pesos mediante regresión Ridge.
• Rubedo (Consolidación): Fija el conocimiento y graba la firma digital neuronal (Watermarking).
GUÍA DE HIPERPARÁMETROS
Radio Espectral (Spectral Radius)
Controla el eigenvalor máximo del reservorio. Valores cercanos o mayores a 1.0 aumentan la "memoria" del sistema a largo plazo pero introducen dinámicas no lineales complejas y caos. Valores bajos (ej: 0.5) estabilizan el sistema pero limitan la memoria temporal.
Tasa de Fuga (Leak Rate)
Define la velocidad de decaimiento del estado interno al asimilar un nuevo estímulo. Una tasa baja (ej: 0.1) mezcla fuertemente el pasado con el presente, ideal para secuencias largas y lentas. Una tasa alta (ej: 1.0) reacciona al instante, descartando la inercia temporal.
Temperatura y Top-P
Hiperparámetros de muestreo de TinyLMv2. La Temperatura amplifica o mitiga las probabilidades de palabras menos comunes (mayor creatividad). El Top-P (Nucleus Sampling) delimita el conjunto de palabras candidatas acumuladas para evitar tokens incoherentes.
